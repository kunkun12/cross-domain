cross-domain
============

introduce two ways to solve cross-domain communication from the browser

#####how to run the demo

		git clone https://github.com/kunkun12/cross-domain.git
		cd cross-domain
		npm install 
		node app.js

type in a URL in browser :

		http://localhost:3002/crossdomain.html
